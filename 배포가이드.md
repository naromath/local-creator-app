# 로컬크리에이터 관리 시스템 — 온라인 배포 가이드

## 시스템 구성

| 구성 요소 | 서비스 | 역할 |
|----------|--------|------|
| 프론트엔드 | **Vercel** | Next.js 앱 호스팅 (자동 빌드/배포) |
| 데이터베이스 | **Supabase** (PostgreSQL) | 데이터 저장, API, 인증 |
| 인증 | **Supabase Auth** | 이메일/비밀번호 로그인 |

---

## 1단계: Supabase 관리자 계정 생성

Supabase 대시보드에서 로그인할 관리자 계정을 만들어야 합니다.

1. https://supabase.com/dashboard 로그인
2. **local-creator** 프로젝트 선택
3. 좌측 메뉴에서 **Authentication** → **Users** 클릭
4. **Add user** → **Create new user** 클릭
5. 관리자 이메일/비밀번호 입력 후 생성
   - 예: `admin@yourcompany.com` / 원하는 비밀번호
   - **Auto Confirm User** 체크

## 4단계: Supabase Redirect URL 설정

Vercel 배포 URL을 Supabase에 등록합니다.

1. Supabase 대시보드 → **Authentication** → **URL Configuration**
2. **Site URL**: `https://your-project.vercel.app`
3. **Redirect URLs**에 `https://your-project.vercel.app/**` 추가

## 비즈니스 규칙 (자동 검증)

- 정부지원금 ≤ 총 사업비의 80%, 최대 4,000만 원

- 2,000만 원 초과 구매 시 비교견적서 필수
- 인증된 사용자만 시스템 접근 가능 (Supabase RLS)
